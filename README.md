## [Smithing MisClick Prevention](https://runelite.net/plugin-hub/show/offer-at-price)

This RuneLite plugin allows users to configure select items in the smithing menu so that left-clicking brings up the right-click menu instead.

The settings can be applied to as many or few items in the smithing menu.
